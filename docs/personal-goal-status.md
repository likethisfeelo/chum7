# personalGoal 현황

## 개요
- `personalGoal`은 참여 시 입력받던 자유 텍스트 다짐(최대 200자)입니다.
- 기존 참여 플로우에서는 "나의 7일 목표" 라벨로 별도 goal 스텝에서 수집되었습니다.

## 저장 위치
- 테이블: `USER_CHALLENGES_TABLE`
- 필드: `personalGoal` (`string | null`)
- 모델: `UserChallenge`

## 현재 상태 (2025 기준)
- 위자드에서 수집 후 DB에 저장됩니다.
- confirm 스텝에서 1회 표시되지만, 참여 완료 이후 사용자 주요 화면에서는 노출되지 않습니다.
- `my-challenges` API 응답에는 포함되어 프론트에서 접근은 가능합니다.

## 위자드 제거 결정
- 수집하지만 보여주지 않는 UX 불일치를 해소하기 위해 참여 위자드의 goal 스텝에서 제거합니다.
- DB 컬럼/API는 유지하여 향후 재활용 가능성을 열어둡니다.

## 향후 옵션
- ME 페이지 챌린지 카드에 "나의 다짐" 한 줄 요약 노출
- 참여 동기 공유 피드 기능으로 확장
- 리더 화면에서 참여자 다짐 모아보기
