# 영상 인증 트림 기능 Phase 4 롤아웃 체크리스트

## 1) 카나리 롤아웃 단계

### 10% 단계

- [ ] dev/stage에서 24시간 이상 무중단 확인
- [ ] `mediaValidationStatus=pending`에서 `valid/invalid` 전환 지연 확인 (p95)
- [ ] 업로드 실패율이 기존 대비 급증하지 않는지 확인

### 30% 단계

- [ ] 10% 단계 알람/오류 대응 이슈 close
- [ ] invalid_media 재업로드 클릭률/성공률 확인
- [ ] 피드 자동 새로고침(15초)로 인한 API 부하 증가 허용 범위 확인

### 100% 단계

- [ ] 30% 단계에서 신규 치명 이슈 없음
- [ ] 운영팀/QA 승인 완료
- [ ] 롤백 경로(기능 플래그/배포 버전) 확인

## 2) KPI 알람 임계치 (초안)

### 업로드/검증

- MediaValidationInvalidCount: **15분 합계 >= 5** → 경고
- MediaValidationErrorCount: **5분 합계 >= 1** → 즉시 경고
- Upload API 실패율: **5분 평균 >= 3%** → 경고

### 재생/피드

- 피드 영상 시작 실패율: **5분 평균 >= 2%** → 경고
- 피드 API p95 latency: **5분 평균 >= 1200ms** → 경고

## 3) 운영 액션

- [ ] 알람 발생 시 최근 1시간 `media_validation_error` 로그 확인
- [ ] `mediaValidationReason` 상위 분포 확인 후 UX 가이드 문구 조정
- [ ] 재업로드 CTA 클릭 후 성공률 저하 시 업로드 네트워크/사이즈 정책 재검토

## 4) 롤백 기준

- [ ] 검증 오류로 인한 사용자 제출 실패가 10분 이상 지속
- [ ] 피드 성능 저하로 CS 이슈 급증
- [ ] 배포 후 치명 결함(데이터 정합성/권한 이슈) 확인
